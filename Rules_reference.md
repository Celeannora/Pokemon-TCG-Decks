# Pokémon TCG rules reference

Quick-reference for AI assistants building decks. This is not a complete rulebook — see the [official Pokémon TCG rulebook](https://www.pokemon.com/us/pokemon-tcg/rules) for full rules.

---

## Game setup

- Each player has a 60-card deck
- Draw 7 cards as opening hand
- Place 1 Basic Pokémon face-down as Active, up to 5 on Bench
- Place top 6 cards face-down as Prize cards
- Win by taking all 6 Prize cards, or opponent has no Bench Pokémon when Active is KO’d, or opponent cannot draw

## Card supertypes

### Pokémon

| Subtype | Description |
|---|---|
| Basic | Can be played directly from hand; required for game start |
| Stage 1 | Evolves from a Basic Pokémon |
| Stage 2 | Evolves from a Stage 1 Pokémon |
| ex / EX | Gives up 2 Prize cards when KO’d (lowercase ex = current era) |
| V | Basic; gives up 2 Prize cards when KO’d |
| VSTAR | Evolves from V; gives up 2 Prize cards; has VSTAR Power (once per game) |
| VMAX | Evolves from V; gives up 3 Prize cards |
| Radiant | Special rarity; only 1 per deck |
| Tera | Has Terastal ability; specific interactions |

### Trainer

| Subtype | Description |
|---|---|
| Supporter | 1 per turn; powerful effects (draw, disruption) |
| Item | Unlimited per turn |
| Stadium | 1 in play at a time; either player can replace |
| Pokémon Tool | Attach to a Pokémon; 1 per Pokémon |
| ACE SPEC | Extremely powerful; only 1 ACE SPEC per deck |

### Energy

| Subtype | Description |
|---|---|
| Basic | 11 types: Grass, Fire, Water, Lightning, Psychic, Fighting, Darkness, Metal, Dragon, Fairy, Colorless. Unlimited copies per deck |
| Special | Provides multiple types or bonus effects. Max 4 copies per deck (by name) |

## Turn structure

1. Draw a card
2. Do any of the following in any order:
   - Play Basic Pokémon to Bench
   - Evolve Pokémon (not first turn played, not first turn of game for going-first player)
   - Attach 1 Energy from hand
   - Play Trainer cards
   - Retreat Active Pokémon (pay retreat cost)
   - Use Abilities
3. Attack (ends your turn)

## Standard format (current)

- **Regulation Marks**: G and later are Standard-legal (G, H, I, J)
- **Basic Energy**: Always legal regardless of regulation mark
- **Rotation**: Occurs annually; older regulation marks rotate out
- **Banned cards**: Check official ban list before building

---

## Key deck building principles

- **Consistency**: Maximize draw and search to see key cards early
- **Recovery**: Include ways to retrieve discarded resources
- **Switching**: Always have outs to move stuck Active Pokémon
- **Prize awareness**: Key cards at 2 or fewer copies risk being prized
- **Evolution timing**: Stage 2 lines need Rare Candy or very consistent search
- **Energy acceleration**: Most competitive decks accelerate energy beyond 1-per-turn
